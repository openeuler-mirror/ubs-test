# 贡献者行为准则

## 我们的承诺

为了营造开放和友好的环境，我们作为贡献者和维护者承诺：让参与我们项目和社区的每个人都能够享受无骚扰的体验，无论其年龄、体型、残疾、种族、民族特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向和性别认同如何。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用性化的语言或图像，以及不受欢迎的性关注或性骚扰
- 恶意评论、侮辱/贬损评论以及人身攻击或政治攻击
- 公开或私下骚扰
- 未经许可发布他人的私人信息，如物理地址或电子邮件地址
- 在专业场合可能被合理认为不适当的其他行为

## 我们的责任

项目维护者有责任阐明可接受行为的标准，并应针对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、问题和其他贡献，并可以在暂时或永久禁止任何他们认为行为不当、威胁、冒犯或有害的贡献者。

## 适用范围

本行为准则适用于项目空间内和公共空间内的所有项目贡献者，当个人代表项目或其社区时。代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体帐户发布或在线上或线下活动中担任指定代表。项目维护者可以进一步定义和阐明项目代表。

## 执行

可以通过项目团队发送电子邮件至 tongjinhui@example.com 来报告滥用、骚扰或其他不可接受的行为。项目团队将审查和调查所有投诉，并将做出他们认为必要且适合情况的回应。项目团队有义务对事件报告者保密。具体执行政策的详细信息可能会单独发布。

不真诚地遵守或执行行为准则的项目维护者可能面临项目领导层确定的其他临时或永久后果。
